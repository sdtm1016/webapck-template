## 运行

1. 安装依赖

   ```shell
   npm install
   ```

2. 打包

   开发：

   ```shell
   npm run build:dev
   ```

   生产：

   ```shell
   npm run build
   ```

3. dll 包

   ```shell
   npm run pack:dll
   ```

4. 包分析

   ```shell
   npm run build --analyze
   ```
